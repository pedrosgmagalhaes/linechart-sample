How to use it

First, clone the repository:
```sh
git clone https://github.com/pedrosgmagalhaes/linechart-sample.git
```
Install it
```sh
yarn install
```
Install required CocoaPods dependencies
```sh
cd ios && pod install
```
Run your simulator
```sh
yarn ios
```

Enjoy it!
